# Brooklyn Bowl Тюмень — статус проекта

Язык общения с пользователем: **русский**.

## Что это
Переделка сайта Brooklyn Bowl (Тюмень) в наш тёмный стиль. Источник контента — страницы
на https://brooklynbowl.bowlsoft.team/ (fetch'им нужную страницу). Отзывы — из реальных
данных (см. `uploads/2025-2026.06. Положительные отзывы для сайта.xlsx`; уже извлечённые
по сценариям лежат в `SCEN_REVIEWS` внутри `Brooklyn Bowl - Главный.dc.html`).

## Дизайн-система (соблюдать на всех страницах)
- Формат: Design Component (`*.dc.html`), inline-стили, без CSS-классов.
- Фон `#0B0B0C`, карточки `#0F0F11`, акцент-красный `#E42725` (hover `#F26B69`), текст `#fff`/`#9C9A95`.
- Шрифты: Proxima Nova / Montserrat (подключены в `<helmet>`).
- Скругл的ения 20–24px, pill-кнопки `border-radius:300px`, hover приподнимает карточку.
- Образец-эталон для сценарных страниц: **`dr.dc.html`** (взрослый ДР).
  Структура: header → hero (бейдж/h1 с красным `<em>` в 0.5em/подзаголовок/2 кнопки/чипы) →
  «что входит» (4 карточки) → форматы/цены (4 карточки, одна красная «чаще берут») →
  координатор → калькулятор (тумблер дня + слайдер гостей + чек-боксы допов + Итого +
  рекомендация пакета Мини/Хит/Макси) → галерея «как выглядит» (6 фото) → отзывы (columns:3,
  «показать ещё») → CTA → модалка-заявка → футер.
- Кешбэк НЕ упоминаем — пишем «двойные бонусы».
- Реальные фото пользователь присылает по ходу; до этого ставим remote с bowlsoft.team.
